Enables you to display the post thumbnails for every post 

== Description ==

	Smart related posts thumbnails plugins generate the thumbnails of the post and give the advantage to user for quick reading of the thumb by showing post excerpt on hover.


== Installation ==
	Installation steps of the plugins as below 
	1. Download the latest verision of smart related post thumbnails form the plugin url:http://wizcrew.com/blog/smart-related-posts-thumbnails-wordpress-plugin/24/
	2. Extract the smart-posts-thumbnails.zip file 
	3. Upload `smart-post-thumbnails` folder  to the `/wp-content/plugins/` directory
	4. Activate the plugin through the 'Plugins' menu in WordPress
	5. Customize plugin settings under Settings -> Smart Related Posts thumbs
